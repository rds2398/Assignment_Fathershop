# How To Run

1. Clone the repository:

   ```bash
   git clone https://github.com/r4rishabh/Vocally-Local.git

   ```

2. Move into the project directory

   ```bash
   cd Assignment_Fathershop

   ```

3. Install dependencies:

   ```bash
   npm install

   ```

4. Run the Development Server
   ```bash
   npm run dev
   ```
